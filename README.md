## 🔨 what i'm building

**[goblog](https://github.com/zulian026/goblog)** — a full-stack blog platform I keep rewriting because I keep learning better ways to do things.

> Started in Laravel + Filament. Migrated to Next.js 15 App Router. Using Prisma v7, Supabase auth, PostgreSQL, Tailwind CSS v4. Deployed on Vercel. Still fighting production bugs on weekends.

Things I've had to figure out the hard way:
- Prisma on Vercel (custom output path, `postinstall` script, pgbouncer on port 6543)
- Supabase SSR + middleware session handling without auto-logout from prefetch
- Role-based auth that doesn't touch Edge Runtime
- Tiptap rich text that doesn't murder your HTML
